Question      
 Quel type d'adressage permet d'assurer la confidentialité des données échangées sur un réseau Ethernet ?
Adressage unicast.
Dans le cadre de l’utilisation conventionnelle d’une carte réseau, seul le destinataire ayant la bonne adresse accepte la trame émise. Cependant, une application peut utiliser le mode promiscuous de la carte pour désactiver cette sélection à la réception. La confidentialité ne peut donc pas être respectée.
Adressage multicast .
Les destinataires de la trame émise s’invitent par eux même pour accepter les trames reçues. Les données ne peuvent être confidentielles puisque ce n’est pas l’objectif de ce mode d’adressage.
Adressage broadcast .
Tous les postes étant destinataires de la trame émise vont accepter les trames. Les données ne peuvent être confidentielles puisque ce n’est pas l’objectif de ce mode d’adressage.
Aucun type d'adressage ne permet d'assurer la confidentialité des données.
Les informations sont diffusées sur toutes les cartes réseaux et ne sont pas cryptées. Le mode promiscuous permet de désactiver le filtrage d’adresse à la réception et d’accepter toutes les trames quel que soit l’adresse de destination utilisée.

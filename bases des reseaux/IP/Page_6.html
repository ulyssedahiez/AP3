Question      
 Que se passe-t-il si une table de table de routage comporte deux adresses IP de passerelle différentes pour une même destination ?
La route la plus courte est choisie.
En l’état, aucune indication ne permet de donner la priorité.
La seconde peut être utilisée si la première est défaillante, on dispose ainsi d'une route de secours.
Aucun mécanisme de backup n’est prévu pour utiliser une autre entrée d’une table de routage, ce qui pourrait conduire à des erreurs de routage.
La même sera toujours utilisée et la seconde ignorée.
Les deux seront utilisées à tour de rôle pour équilibrer la répartition du trafic.
Aucun mécanisme d’équilibrage de charge n’est implémenté dans une table de routage. Une seule route est éligible.

Adressage IP - Protocole IP

Cours  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           


Le protocole IP ajoute un niveau d'adressage supplémentaire destiné à gérer l'unicité d'une adresse dans un standard mondial.



Pour cela, l'ICANN, organisme international, gère l'allocation des adresses du protocole IP.

Dans ce cours, le protocole IP est abordé dans sa version 4. Cette version initialement conçue pour fédérer toutes les ressources s'est avérée limitée par son champ d'adressage sur quatre octets.

Ce protocole reste cependant toujours d'actualité.

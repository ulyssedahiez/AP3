Définitions - Protocole IP

Définitions  
IP

Internet Protocol

Simples

Transmettre des données en quantité limitée, d’une application à une autre, en contrôlant l’intégrité par un checksum.

Loopback

Cette interface correspond à l0 sous linux, alors qu’elle n’apparait pas explicitement sous windows, elle est cependant toujours présente.

Datagrammes

Les messages issues des couches supérieures sont encapsulées par le protocole IP et constituent des datagrammes.

La notion de trame est plutôt réservée au protocole Ethernet.

Format hexdump

Dans ce format, les données sont représentées en hexadécimal sous forme d’une suite octets, Chaque ligne d’octets est précédée par une adresse indiquant sa position relative dans le fichier.

Paramètre

Le payload est remis à la couche supérieure. Dans certains cas, des paramètres complémentaires peuvent y être associés.

Réponse

Pour simuler un échange de message, une réponse ICMP est renvoyée suite à la réception d’une requête ICMP.

D’interface réseau

A chaque carte réseau correspond une interface dans le système d’exploitation du poste.

Requête

Pour simuler un échange de message, une requête ICMP permet de solliciter un poste.

Couche réseau

Elle fait référence à la couche n°3 du modèle OSI.

Voir chapitre "encapsulation".

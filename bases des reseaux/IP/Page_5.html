Routage local - Protocole IP

Cours  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           


Pour compléter les fonctions de la couche réseau dont fait partie le protocole IP, le routage IP intervient dans la distribution des datagrammes pour les orienter vers la bonne destination sans inonder les réseaux non concernés.



Il intervient en tout premier lieu pour la distribution des messages sur un réseau local.

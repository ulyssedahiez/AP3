Syntaxe de génération de trames - Manipulation de protocoles

Trame Ethernet 802.3  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
variable_python = Dot3(dst="@ethernet" src="@ethernet" len=length)

format des paramètres
@ethernet = xx:xx:xx:xx:xx:xx
length = nombre d'octets du payload (indiquez 0 pour ne pas inclure de trame 802.2 supplémentaire dans le payload)

Exemple
trame_ethernet= Dot3(dst="00:0f:66:56:fa:d2", src="00:ae:f3:52:aa:d1", len=0)

Trame Ethernet II  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
variable_python = Ether(dst="@ethernet" src="@ethernet" type=ether_type)

format des paramètres
@ethernet = xx:xx:xx:xx:xx:xx
ether_type = 0x . . . . ou "protocole"

Exemples
trame_ethernet= Ether(dst="00:0f:66:56:fa:d2", src="00:ae:f3:52:aa:d1", type=0x800)

format des paramètres
@IP = xxx.xxx.xxx.xxx

Trame ARP  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
variable_python = ARP(hwsrc="@ethernet, psrc="@IP", hwdst="@ethernet, pdst="@IP", op=operation)

format des paramètres
@ethernet = xx:xx:xx:xx:xx:xx
@IP = xxx.xxx.xxx.xxx
operation = 0x01 ou 0x02

Exemple
trame_ip= ARP(hwsrc="01:02:0:04:05:06", psrc="192.168.1.1", hwdst="07:08:09:0a:0b:0c", pdst="192.168.1.1", op=0x01)

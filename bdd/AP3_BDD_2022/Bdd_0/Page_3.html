SQL - Base de données relationnelle

SQL  
Compréhension :   0   0   0   0   0   0     QCM :   0   0   0           
L'élaboration de requêtes SQL nécessite le suivi d'un protocole précis sans lequel vous pouvez obtenir des résultats qui n'ont aucun rapport avec ce qui est demandé.

Démarche :

- Identifiez sur le MCD toutes les entités et les relations concernées par la requête

- Identifiez les tables correspondantes à ces entités et à ces relations

- Restreignez le nombre de tables au strict minimum en fonction des données demandées

- Construisez la requête

- Testez la requête sur un nombre restreint de données judicieusement choisies pour éviter les "fausses vérifications"

- Vérifiez manuellement ce que vous devriez obtenir comme données

Téléchargement du tutoriel

Installation du cours sur votre disque local:

  1. Créez un répertoire cible (p.ex: COURS-C)
  2. Téléchargez le fichier comprimé et copiez-le dans le nouveau répertoire
  3. Décomprimez le fichier dans le nouveau répertoire
  4. Chargez le fichier de départ: "home.htm" (que vous pouvez aussi ajouter à la liste de vos sites préférés)
